# high-tone

自爪育成サロン **high‑tone** の公式ウェブサイトです。
HTML / CSS / JavaScript を用いて静的サイトとして構築されています。

---

## 🧩 概要

このリポジトリには以下のファイル・フォルダが含まれています：

```
├── css/            # スタイルシート（SCSS をコンパイルした CSS）
├── sass/           # SCSS ソースコード
├── images/         # 画像ファイル
├── index.html      # トップページ
├── menu.html       # メニューページ
├── script.js       # JavaScript 実装
└── README.md       # 本ドキュメント
```

---

## 🚀 このサイトについて

**high‑tone** は自爪育成を目的とした美容サロンで、
以下の情報を提供しています：


- サロンについて（コンセプト）
- 施術事例（ビフォー・アフター）
- 選ばれる理由
- メニュー内容
- 料金案内
- 施術の流れ
- ご予約・お問い合わせ

  （GitHub Pages / Netlify / Vercel 等）

## 📦 開発方法

### SCSS の編集

`sass/` ディレクトリ内で SCSS を管理しています。
編集後、CSS にコンパイルしてください。

```bash
sass --watch sass:css
```

---

## 👀 プレビュー方法

ローカルで確認する場合は、`index.html` を直接ブラウザで開くか、
簡易サーバーを使用してください。

```bash
python3 -m http.server
```
